Entscheidungsmatrix

Kriterium A: Rich Panel B: Inline Edit C: Hybrid
Textformatierung (Farbe, Bold) Tiptap im Panel Floating Toolbar Beides
Repeater (FAQ, Cards) Voll im Panel Schwierig inline Panel für Repeater
Nicht-Text-Felder (URLs, Bilder) Im Panel Im Panel Im Panel
Live-Preview Panel → Seite Direkt Beides sync
Entwicklungsaufwand Mittel Hoch Mittel-Hoch
Übersicht aller Felder Immer sichtbar Nur aktives Feld Immer sichtbar

Meine Empfehlung: Option A als Basis bauen (Rich Editor im Panel, alle Textfelder mit Tiptap + Farbpalette + Free Color Picker). Dann Option C als Erweiterung (Klick auf Seite → Focus auf Panel-Feld). Option B (Inline-only) ist riskant wegen contenteditable Bugs und schwierigem Repeater-Handling.
